# Common

## EnumClass

Шаблонный класс, приблизительно имитирующий Java enum.

## LocalDate

Утилиты для &lt;chrono>.

## Numeric

Утилиты для &lt;boost/decimal.hpp>.

## ObservableValue

Шаблонный класс, основанный на классе с таким же именем из JavaFX.
Реализован простейший вариант подписки на изменение значения.
